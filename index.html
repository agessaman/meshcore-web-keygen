MeshCore Key Generator

Generate custom Ed25519 keys for MeshCore

About MeshCore Keys

  • Meshcore uses the first two characters of your public key as a node identifier. It is helpful to be able to select your node id to avoid collisions with neighbors.
  • This key generator does all processing in your browser - keys never leave your device

📋 How to Import Keys into MeshCore

×

🔧 For Companion Nodes

  1. Connect to your node using the MeshCore app
  2. Tap the Settings gear icon
  3. Tap Manage Identity Key
  4. Paste your Private Key into the text box
  5. Tap Import Private Key
  6. Important: Tap the checkmark ✓ to save changes in settings

📡 For Repeater Nodes

⚠️ Important: You must temporarily flash companion firmware first.
  1. Flash your node with companion firmware temporarily
  2. Follow the companion instructions above
  3. After importing, you can safely re-flash to repeater firmware

📄 Alternative: JSON Import

If you downloaded the JSON file from this keygen:

  1. In the MeshCore app settings, tap Import Config
  2. Select your downloaded JSON file
  3. The public and private keys will be automatically imported

💡 Tips

  • Make sure to copy the entire private key (128 characters)
  • The new public key won't appear until you save the settings
  • Keep your private key secure and never share it
Enter any hex prefix (1-8 characters). Longer prefixes take longer to find.
